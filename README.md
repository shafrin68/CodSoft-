# CODSOFT AI Internship - Task 1

## Chatbot with Rule-Based Responses

### Overview
This project is a simple rule-based chatbot developed as part of the CODSOFT AI Internship. The chatbot interacts with users by recognizing predefined inputs and providing appropriate responses.

### Features
- User-friendly conversation
- Predefined responses for common questions
- Greeting and farewell messages
- Simple and beginner-friendly implementation

### Technologies Used
- Python

2. Open the project folder.
